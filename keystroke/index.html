Keystroke Patterns as Prosody in Digital Writings:
A Case Study with Deceptive Reviews and Essays
Dataset
  • Downloads:
    • Restaurant Reviews: [download]
    • Gay Marriage: [download]
    • Gun Control: [download]

  • File Format: tab seperated values

  • Data Fields: [UserName, AccessKey, Topic, Opinion, Date, ReviewType, Task, Group, Flow, ReviewText, ReviewMeta]

    • ReviewMeta field has a list of KeyUP, KeyDown, and MouseUp event information including timestamps since the first user input event on the task page. First event should have timestamp of 0, but sometimes it starts with a different value. Still, the difference between the timestamps of one event and the first event will show the corrent timestamp.

      • KeyUp/KeyDown: <timestamp> KeyUp/KeyDown <KeyCode>
          (* KeyCode is a javascript keycode)
      • MouseUp: <timestamp> MouseUp <start_index> <end_index>
        start and end indices tell you the selected text range. These values will help you the editing behavior of the writer.
Paper
Keystroke Patterns as Prosody in Digital Writings: A Case Study with Deceptive Reviews and Essays
- Ritwik Banerjee, Song Feng, Jun Seok Kang, Yejin Choi, EMNLP 2014
