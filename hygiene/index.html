Where Not to Eat? Improving Public Policy by Predicting Hygiene Inspections Using Online Reviews
Dataset (classification instances based on "inspection periods" as used in the paper)
  • Data set (per-inspection period, aggregated reviews & other meta data)
    • Download: here
    • File Format: comma seperated values
    • Data Fields: [inspection_id, restaurant_id, inspection_period_start_date, inspection_period_end_date,
      inspection_penalty_score, inspection_average_prev_penalty_scores, inspection_prev_penalty_score,
      cuisines, zip_code, review_count, non_positive_review_count, average_review_rating, review_contents]
Additional Data (potentially useful records and data in addition to above)

  • Restaurant Meta Data

  • Reviews
